# 教室借用查询接口文档

## 接口概览
- **接口地址**：`http://202.119.81.112:9080/njlgdx/kbxx/jsjy_query2`
- **请求方式**：`POST`
- **返回格式**：`HTML`
- **认证方式**：Cookie 需携带有效 `JSESSIONID`（示例：`JSESSIONID=9934169FC5F555**0BA02046AD7C96EE`）

---

## 请求参数说明

| 参数名     | 类型   | 必填 | 说明                                                                 |
|------------|--------|------|----------------------------------------------------------------------|
| typewhere  | string | 是   | 固定值：`jszq`（教室占用）                                           |
| xnxqh      | string | 是   | 学年学期号，示例：`2025-2026-2`                                      |
| xqbh       | string | 是   | 校区编号：<br>`4y` → 江阴校区<br>`01` → 孝陵卫校区               |
| jxqbh      | string | 否   | 教学区编号（无效字段，请留空）                                      |
| jxlbh      | string | 否   | 教学楼编号（无效字段，请留空）                                      |
| jsbh       | string | 否   | 教室编号（无效字段，请留空）                                        |
| bjfh       | string | 否   | 班级符号（无效字段，请留空）                                        |
| rnrs       | string | 否   | 容纳人数，留空表示不限                                               |
| jszt       | string | 是   | 教室状态：<br>`5` 空闲<br>`7` 临时调课<br>`6` 固定调课<br>`4` 考试<br>`3` 锁定<br>`2` 借用<br>`1` 正常上课<br>留空返回全部 |
| zc         | string | 是   | 起始周次，示例：`1`（第1周）                                         |
| zc2        | string | 是   | 结束周次，示例：`2`（第2周）                                         |
| xq         | string | 是   | 起始星期，示例：`1`（星期一）                                       |
| xq2        | string | 是   | 结束星期，示例：`3`（星期三）                                       |
| jc         | string | 否   | 起始节次，留空表示不限                                               |
| jc2        | string | 否   | 结束节次，留空表示不限                                               |

---

## 使用示例
若需查询**孝陵卫校区**本学期第1周至第2周、周一至周日全部节次的**空闲教室**，请按如下条件请求：

- 周次范围：第1周 ～ 第2周
- 星期范围：周一 ～ 周日
- 节次范围：留空（不限）
- 教室状态：`jszt=`（留空返回全部）

随后系统会返回给您一个表，表中未被填充任何元素的即为空闲，其他填充： Ｌ 临时调课; Ｇ 固定调课; 空闲; Κ 考试; Ｘ 锁定; Ｊ 借用; ◆ 正常上课;


## 请求示例

### curl
```bash
curl -X POST "http://202.119.81.112:9080/njlgdx/kbxx/jsjy_query2" \
     -H "Cookie: JSESSIONID=YOUR_SESSION_ID" \
     -H "Content-Type: application/x-www-form-urlencoded" \
     --data-urlencode "typewhere=jszq" \
     --data-urlencode "xnxqh=2025-2026-2" \
     --data-urlencode "xqbh=01" \
     --data-urlencode "jszt=" \
     --data-urlencode "zc=1" \
     --data-urlencode "zc2=2" \
     --data-urlencode "xq=1" \
     --data-urlencode "xq2=7" \
     --data-urlencode "jxqbh=" \
     --data-urlencode "jxlbh=" \
     --data-urlencode "jsbh=" \
     --data-urlencode "bjfh=" \
     --data-urlencode "rnrs=" \
     --data-urlencode "jc=" \
     --data-urlencode "jc2="
```

### Python
```python
import requests

url = "http://202.119.81.112:9080/njlgdx/kbxx/jsjy_query2"
headers = {
    "Cookie": "JSESSIONID=YOUR_SESSION_ID",
    "Content-Type": "application/x-www-form-urlencoded"
}

data = {
    "typewhere": "jszq",
    "xnxqh": "2025-2026-2",
    "xqbh": "01", # 01 为孝陵卫校区
    "jszt": "",   # 留空返回全部
    "zc": "1",    # 第1周
    "zc2": "2",   # 第2周
    "xq": "1",    # 周一
    "xq2": "7",   # 周日
    "jxqbh": "",
    "jxlbh": "",
    "jsbh": "",
    "bjfh": "",
    "rnrs": "",
    "jc": "",
    "jc2": ""
}

response = requests.post(url, headers=headers, data=data)
print(response.text)
```
