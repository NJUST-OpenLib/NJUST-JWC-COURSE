# 教室借用查询接口文档

## 接口概览
- **接口地址**：`http://202.119.81.112:9080/njlgdx/kbxx/jsjy_query2`
- **请求方式**：`POST`
- **返回格式**：`HTML`
- **认证方式**：Cookie 需携带有效 `JSESSIONID`（示例：`JSESSIONID=9934169FC5F555**0BA02046AD7C96EE`）

---

## 请求参数说明

| 参数名     | 类型   | 必填 | 说明                                                                 |
|------------|--------|------|----------------------------------------------------------------------|
| typewhere  | string | 是   | 固定值：`jszq`（教室占用）                                           |
| xnxqh      | string | 是   | 学年学期号，示例：`2025-2026-2`                                      |
| xqbh       | string | 是   | 校区编号：<br>`4y` → 江阴校区<br>`01` → 孝陵卫校区               |
| jxqbh      | string | 否   | 教学区编号（无效字段，请留空）                                      |
| jxlbh      | string | 否   | 教学楼编号（无效字段，请留空）                                      |
| jsbh       | string | 否   | 教室编号（无效字段，请留空）                                        |
| bjfh       | string | 否   | 班级符号（无效字段，请留空）                                        |
| rnrs       | string | 否   | 容纳人数，留空表示不限                                               |
| jszt       | string | 是   | 教室状态：<br>`5` 空闲<br>`7` 临时调课<br>`6` 固定调课<br>`4` 考试<br>`3` 锁定<br>`2` 借用<br>`1` 正常上课<br>留空返回全部 |
| zc         | string | 是   | 起始周次，示例：`1`（第1周）                                         |
| zc2        | string | 是   | 结束周次，示例：`2`（第2周）                                         |
| xq         | string | 是   | 起始星期，示例：`1`（星期一）                                       |
| xq2        | string | 是   | 结束星期，示例：`3`（星期三）                                       |
| jc         | string | 否   | 起始节次，留空表示不限                                               |
| jc2        | string | 否   | 结束节次，留空表示不限                                               |

## 使用示例

### 示例

http://202.119.81.112:9080/njlgdx/kbxx/jsjy_query2?typewhere=jszq&xnxqh=2025-2026-2&xqbh=01&jxqbh=&jxlbh=&jsbh=&bjfh=&rnrs=&jszt=5&zc=1&zc2=2&xq=1&xq2=7&jc=&jc2=


请求后，系统返回的 HTML 表格中：  
- 空白格 → 空闲  
- 字符含义对照：  
  L＝临时调课，G＝固定调课，Κ＝考试，X＝锁定，J＝借用，◆＝正常上课  


## 优化请求策略 (推荐)

为减少对教务系统服务器的压力并防止 IP 被封禁，建议采用 **“按周查询，本地拆分”** 的策略：

1.  **单次请求整周**：将 `xq` 设为 `1`，`xq2` 设为 `7`。
2.  **本地解析**：在返回的 HTML 中，通过解析 `id="dataList"` 的表格，根据表头中的 `colspan` 属性将数据按星期拆分。
3.  **批量获取**：循环周次（1-19 周），每次请求间隔建议在 **2秒以上**。

## 数据格式说明

解析后的数据建议保存为 JSON 格式，结构示例：

```json
{
  "week": 1,
  "day": 1,
  "day_name": "星期一",
  "data": [
    {
      "room_name": "I-101",
      "schedule": [
        {
          "slot": "第一大节",
          "is_free": true,
          "occupancy_type": null
        },
        {
          "slot": "第二大节",
          "is_free": false,
          "occupancy_type": "◆"
        }
      ]
    }
  ]
}
```

## 可视化工具

项目提供了配套的可视化预览工具 `index.html`。

### 使用方法：
1.  启动本地静态服务器（如 `python -m http.server 3000`）。
2.  在浏览器中通过 `data` 参数引用 JSON 文件：
    `http://127.0.0.1:3000/document/空闲教室/index.html?data=/document/空闲教室/output/第1周-星期1-classroom.json`
