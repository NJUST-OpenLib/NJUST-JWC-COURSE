# NJUST-JWC-COURSE (南京理工大学教务系统成绩与课表查询工具)

这是一个基于 Python 的南京理工大学（NJUST）强智教务系统（JWC）信息查询脚本。它可以自动登录并抓取学生的学期成绩和课程表信息。

## 功能特点

- **自动登录**：支持通过学号和密码登录教务系统。
- **凭据保存**：可选保存账号密码，在 Cookie 失效时自动尝试重新登录，无需反复输入。
- **会话持久化**：成功登录后会将 `JSESSIONID` 保存到 `.env` 文件中，下次运行可自动恢复会话。
- **主菜单交互**：运行脚本后进入主菜单，可自由选择查询成绩、查询课表或切换账号。
- **成绩查询**：获取并格式化展示所有学期的考试成绩。
- **课表查询**：
    - 默认显示最新学期的课表。
    - 支持交互式选择历史学期进行查询。
    - 自动解析课程名称、教师、时间、学分、地点等详细信息。
- **切换账号**：支持一键清除保存的登录信息并切换其他账号登录。

## 技术说明

本项目针对南京理工大学所使用的**强智科技教务系统**。由于该系统版本较老，并未提供现代化的 JSON API 接口，因此脚本的所有数据获取均通过模拟浏览器请求并对返回的 **HTML 页面**进行解析实现。

- 使用 `requests` 库进行网络请求和 Cookie 管理。
- 使用 `BeautifulSoup` (bs4) 和正则表达式 (`re`) 对 HTML 页面进行解析和提取。

## 安装与运行

1. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```

2. **运行脚本**：
   ```bash
   python main_new.py
   ```
   首次运行需输入学号和密码。成功登录后，脚本会在本地生成 `.env` 文件保存会话。
